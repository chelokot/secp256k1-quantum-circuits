# Generated results

These files are generated by the repository verification and research commands.

## Quick path

- `repo_verification_summary.json`
- `exact_archive_verification_summary.json`

Regenerate with:

```bash
python scripts/verify_all.py
```

## Strict path

- `strict_verification_summary.json`

Regenerate with:

```bash
python scripts/verify_strict.py --mode all
```

The strict summary aggregates:

- lookup contract audit,
- scaffold replay,
- extended toy-family proof,
- projection sensitivity,
- meta-analysis,
- claim boundary matrix.

## Research path

- `research_pass_summary.json`
- `literature_matrix.json`
- `physical_stack_reference_points.json`

Regenerate with:

```bash
python scripts/run_research_pass.py
```

The research pass aggregates:

- dominant cost breakdown,
- literature-inspired scenario bands,
- deterministic secp-family challenge ladder,
- literature and physical-stack reference matrices.
