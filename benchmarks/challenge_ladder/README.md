# Challenge ladder

This directory contains a deterministic family of tiny benchmark curves in the
same `y^2 = x^3 + 7` family as secp256k1.

Files are regenerated by:

```bash
python scripts/run_research_pass.py
```

Contents:

- `challenge_ladder.json` — curve definitions, subgroup generators, and known challenge points
- `challenge_ladder_audit.csv` — end-to-end scalar-accumulation replay transcript
- `challenge_ladder_summary.json` — hash and summary information

See `docs/CHALLENGE_LADDER.md` for interpretation and scope.
